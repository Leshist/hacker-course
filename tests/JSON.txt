Вопрос 6
Какой из примеров JSON верный?

Варианты
( "name" : "John", "age" : 15 )
{ name : "John", age : 15 }
[ "name" : "John", age : "15" ]
{ "name" : "John", "age" : 15 }

Вопрос 7
Какой из примеров JSON верный?

Варианты
{ "id" : 123, "things" : ["a" : 1, "b" : 1] }
[ "id" = 123, "things" = ["a" = 1, "b" = 1] ]
{ "id" : 123, "things" : { "a" : 1, "b" : 1 } }
{ "id" : 123, "things" : [ ( "a" : 1 ) , ( "b" : 1 ) ] }

Вопрос 8
Какой из примеров JSON верный?

Варианты
{ "id" : "1234" ,  "list" : [ 1,  2,  3 ] }
{ id : 1234 ,  list : [ 1,  2,  3 ] }
{ "id" : "1234" ;  "list" : [ "1" ; "2" ; "3" ] }
{ "id" : "1234" ,  "list" : { 1,  2,  3 } }

Вопрос 9
Какой из примеров JSON верный?

Варианты
{ "a" : "null", "b" : [ "c" : "null" ] }
{ "a" : "null", [ "b" :  {"c" : "null" } ] }
{ "a" : null, "b" : [ { "c" : null } ] }
{ "a" : null, "b" : [ "c" : null ] }
